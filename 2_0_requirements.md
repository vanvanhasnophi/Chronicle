# Chronicle 2.0 需求清单

本文件为独立于 2_0_plan.md 的结构化需求列表。每个需求包含唯一 ID，格式说明：
- 前缀 `N` 表示“新功能（New）”，`O` 表示“其他优化（Other）”；
- 形如 `N-1-2` 表示“新功能”下第1类的第2条需求。

---

## 新功能（N）

### N-1 追忆（Recollection）
- N-1-1：**追忆卡片 CRUD**：支持创建、编辑、删除追忆卡片，并为卡片添加元数据（标题、日期、来源）。
- N-1-2：**卡片标签与搜索**：卡片支持标签、全文搜索与按标签/时间筛选。
- N-1-3：**媒体支持**：卡片可以嵌入图片、音频、视频与其他媒体并在卡片内播放/预览。
- N-1-4：**前端追忆 Tab**：在前端新增“追忆”Tab，用时间线、堆叠或卡片墙视图显示卡片。
- N-1-5：**卡片聚合与过滤**：支持按时间、标签、关联（人物/地点）聚合与筛选，支持收藏与排序。
- N-1-6：**导入/导出**：支持将追忆卡片导入/导出为 Markdown（包含媒体引用或附件清单）。

### N-2 页面编辑器（Workdown Pages）
- N-2-1：**可视化页面设计器**：提供拖拽布局、列/行/块模型与可配置容器。
- N-2-2：**可复用组件与模板**：支持组件库、模板保存与应用。
- N-2-3：**高级视觉效果**：支持动画、分层、遮罩、背景媒体与模糊等效果配置。
- N-2-4：**设为主页与预览**：允许将自定义页面设为主页，并提供实时预览与导出功能。
- N-2-5：**页面版本与回退**：记录页面历史版本，支持比较与回退。

### N-3 UI 主题（LaF）与时效性内容主题（Content Theme）
- Completed N-3-1：**字体配置（前后台分离）**：支持默认语言与字体模块，前台/后台可分别设置字体。
- Completed N-3-2：**配色与深浅模式**：支持站点深/浅模式、强调色与背景色配置。
- N-3-3：**动态背景**：支持全局背景图片/视频配置及背景模糊度调整。
- N-3-4：**主题打包与导入导出**：支持将主题打包、导出与导入安装。
- N-3-5：**内容主题绑定**：支持将主题与特定文章或标签绑定并通过主页入口直达。
- N-3-6：**主题时效性与调度**：支持基于时间窗口（年/季/月）自动应用主题到全站或指定内容集。

## 其他优化（O）

### O-1 文章历史版本 & 本地备份
- O-1-1：**编辑中自动备份**：在编辑过程中与保存时自动生成备份快照。
- O-1-2：**历史查看与差异**：支持查看历史版本列表、差异对比并恢复至任意历史版本。
- O-1-3：**整站/文章导出与恢复**：支持导出单篇或整站为本地备份并提供恢复流程。

### O-2 Markdown 编辑器优化
- O-2-1：**缩进**：捕获 Tab 键用于缩进（输入制表符），段前空格识别为制表符用于缩进段落，保持排版灵活性。
- O-2-2：**列表硬/软标签区分**：输入显式序号（如 `1.`, `a.`）视为硬有序标签；否则使用软有序标签接续上文；无序列表与有序列表独立编号规则。
- O-2-3：**编辑器/预览对齐**：编辑器与预览窗格自动对齐滚动，编辑时光标附近内容在预览中高亮。
- O-2-4：**自动格式**：换行或缩进时自动补齐缩进与列表序号（默认软标签），调整列表层级时自动修正序号。
- O-2-5：**内嵌公式（KaTeX）**：支持行内与块级 KaTeX 渲染，公式可用于引用、图表与图片标题。
- O-2-6：**真实预览（Full Preview）**：编辑器与管理页面使用与前台一致的渲染引擎和样式进行完整预览。
- O-2-7：**文章属性面板**：在编辑器 UI 中暴露文章属性（Tag、别名、标题、阅读偏好等），支持实时修改。
- O-2-8：**文章别名**：支持为文章配置友好别名用于可读链接和 SEO（提供示例映射规则）。
- O-2-9：**查找替换覆盖浏览器**：在编辑器内拦截 `Ctrl/Cmd+F` 调出内置查找替换悬浮框，支持正则与大小写匹配。
- O-2-10：**保存/发布快捷键**：拦截 `Ctrl/Cmd+S` 打开保存/发布对话框（操作合并但按钮区分）。
- O-2-11：**导出/打印快捷键**：拦截 `Ctrl/Cmd+P` 调出专用导出/打印对话框。

### O-3 阅读体验
- O-3-1：**长文按需加载**：对超长文章实现分段懒加载或分页以提升性能。
- Completed O-3-2：**TOC 与地址栏锚点**：将目录（TOC）集成到 URL 锚点，实现直达标题的分享与跳转。
- O-3-3：**移动端轻量解析**：移动端使用简化解析与渲染，剔除不必要组件以提升流畅度。

### O-4 搜索体验
- O-4-1：**多维度搜索**：支持按文章全文、文章别名、时间范围与标签组合搜索与排序。
- O-4-2：**Tag 智能筛选**：在搜索框中输入 tag 名称时提供 tag 自动补全与直接以 tag 筛选结果。
